## lumen swagger api json lib

### 在composer.json文件中加入
```
{
    "repositories": [
        {"type": "git", "url": "git@git.util.ahaschool.com:ahamkt/aha-services.git", "reference":"master"}
    ],
    "require": {
        "ahaschool/services": "*"
    }
}
```

### 在运行composer跟新依赖
```
composer install
```
